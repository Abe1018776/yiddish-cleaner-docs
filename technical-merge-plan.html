yiddish-cleaner (DESTINATION)

Laravel 12 app with Inertia v2/Vue 3 frontend. Has audio samples, transcriptions, cleaning pipeline, ASR, alignment, benchmarking. Deployed via Herd.

Stack: PHP 8.4, Laravel 12, Vue 3, Tailwind v4, SQLite, Spatie Media Library, Fortify auth, Reverb WebSockets

2.

yiddish-mapping (SOURCE)

Single-page vanilla JS app for mapping audio files to transcript files. Uses a static data.json with 4669 audio and 1065 transcript records from Google Drive. Deployed to Cloudflare Pages.

Stack: Vanilla HTML/CSS/JS, static data.json, localStorage for user mappings

3.

yiddish-audio-alignment (SOURCE)

Python Flask app for cleaning Yiddish transcripts and word-level audio alignment using stable-ts with yi-whisper. Has RunPod serverless GPU support. Branch: feature/stable-ts-alignment.

Stack: Python 3.11, Flask, stable-ts, faster-whisper, RunPod API, Google APIs

This is a single-file application with no build step. All logic is inline JavaScript inside public/index.html. Key features:

  • 4 tabs: 50-Hour Selection, All Matched, Unmatched Audio, Unmatched Transcripts
  • Suggested matches with confidence scores per unmatched item
  • Manual mapping saved to localStorage
  • CSV export of manual mappings
  • Filters: year (Hebrew calendar), month, type (sicha/maamar/farbrengen), text search
  • Pagination: 50 items per page
  • Modal for bidirectional search (audio-to-transcript and transcript-to-audio)

Controller & Routes
New Controller: app/Http/Controllers/MappingController.php
<?php
namespace App\Http\Controllers;

use App\Models\MappingEntry;
use Illuminate\Http\Request;
use Inertia\Inertia;

class MappingController extends Controller
{
    public function index(Request $request)
    {
        $tab = $request->get('tab', 'selected');

        $stats = [
            'totalAudio' => MappingEntry::whereNotNull('audio_name')->distinct('audio_name')->count(),
            'totalTranscripts' => MappingEntry::whereNotNull('transcript_name')->distinct('transcript_name')->count(),
            'matched' => MappingEntry::where('status', 'matched')->count(),
            'selected50hr' => MappingEntry::where('status', 'selected')->count(),
            'unmatchedAudio' => MappingEntry::where('status', 'unmatched_audio')->count(),
            'unmatchedTranscripts' => MappingEntry::where('status', 'unmatched_transcript')->count(),
        ];

        $query = MappingEntry::query()
            ->when($request->year, fn($q, $v) => $q->where('year', $v))
            ->when($request->month, fn($q, $v) => $q->where('month', $v))
            ->when($request->type, fn($q, $v) => $q->where('type', $v))
            ->when($request->search, fn($q, $v) => $q->where(function($q2) use ($v) {
                $q2->where('audio_name', 'like', "%{$v}%")
                   ->orWhere('transcript_name', 'like', "%{$v}%")
                   ->orWhere('first_line', 'like', "%{$v}%");
            }));

        // Filter by tab
        $statusMap = [
            'selected' => 'selected',
            'matched' => 'matched',
            'unmatched' => 'unmatched_audio',
            'transcripts' => 'unmatched_transcript',
        ];
        $query->where('status', $statusMap[$tab] ?? 'selected');

        return Inertia::render('Mapping/Index', [
            'entries' => $query->paginate(50)->withQueryString(),
            'stats' => $stats,
            'filters' => $request->only(['tab', 'year', 'month', 'type', 'search']),
        ]);
    }

    public function map(Request $request, MappingEntry $entry)
    {
        $validated = $request->validate([
            'matched_name' => 'required|string',
            'matched_link' => 'nullable|string',
        ]);

        $entry->update([
            'status' => 'manually_mapped',
            'transcript_name' => $entry->status === 'unmatched_audio'
                ? $validated['matched_name'] : $entry->transcript_name,
            'audio_name' => $entry->status === 'unmatched_transcript'
                ? $validated['matched_name'] : $entry->audio_name,
            'mapped_by' => $request->user()->id,
            'mapped_at' => now(),
        ]);

        return back();
    }

    public function export(Request $request)
    {
        // CSV export of manual mappings
        $entries = MappingEntry::where('status', 'manually_mapped')->get();
        // ... stream CSV response
    }

    public function searchTranscripts(Request $request)
    {
        return MappingEntry::whereNotNull('transcript_name')
            ->where('transcript_name', 'like', '%' . $request->q . '%')
            ->limit(100)
            ->get(['transcript_name', 'transcript_link', 'year', 'month', 'day']);
    }

    public function searchAudio(Request $request)
    {
        return MappingEntry::whereNotNull('audio_name')
            ->where('audio_name', 'like', '%' . $request->q . '%')
            ->limit(100)
            ->get(['audio_name', 'audio_link', 'year', 'month', 'day', 'est_minutes']);
    }
}

Architecture decision: The yiddish-cleaner app already has an AlignmentManager with a driver pattern. We will add a new StableTsDriver that calls the RunPod serverless endpoint (same as the Python Flask app does), and optionally a LocalStableTsDriver that calls a local Python script via subprocess. This is the exact same pattern as the existing WhisperXDriver and LocalWhisperXDriver.
Important: The heavy ML dependencies (stable-ts, faster-whisper, torch ~2GB) should NOT be installed on the Laravel server. The primary path is RunPod serverless GPU. The local Python scripts are only for development/testing without RunPod.
New StableTsDriver (RunPod)
app/Services/Alignment/Drivers/StableTsDriver.php
<?php
namespace App\Services\Alignment\Drivers;

use App\Services\Alignment\AlignedWord;
use App\Services\Alignment\AlignmentDriverInterface;
use App\Services\Alignment\AlignmentResult;
use Illuminate\Support\Facades\Http;
use Illuminate\Support\Facades\Log;
use Illuminate\Support\Facades\Storage;
use RuntimeException;

/**
 * Stable-TS alignment driver using RunPod serverless.
 *
 * Uses the ivrit-ai confusion-aware algorithm with yi-whisper
 * for high-quality Yiddish forced alignment.
 */
class StableTsDriver implements AlignmentDriverInterface
{
    protected string $endpointId;

    public function __construct(
        protected ?string $apiKey,
        protected string $model = 'ivrit-ai/yi-whisper-large-v3-turbo-ct2',
    ) {
        $this->endpointId = config('alignment.providers.stable_ts.endpoint_id', '');
    }

    public function align(string $audioPath, string $text, array $options = []): AlignmentResult
    {
        if (! $this->apiKey || ! $this->endpointId) {
            throw new RuntimeException('RunPod API key and endpoint ID required for stable-ts alignment');
        }

        // Upload audio to temporary public URL (S3/R2)
        $audioUrl = $this->uploadAudioForRunPod($audioPath);

        try {
            $result = $this->callRunPod([
                'mode' => 'align',
                'audio_url' => $audioUrl,
                'text' => $text,
                'language' => $options['language'] ?? 'yi',
            ]);

            return $this->buildResult($result, $text);
        } finally {
            // Clean up temporary upload
            $this->cleanupUpload($audioUrl);
        }
    }

    // ... RunPod polling, result parsing, etc.
    // Port from runpod_client.py
}

scripts/alignment/run_alignment.py (subprocess entrypoint)
#!/usr/bin/env python3
"""
CLI entrypoint for stable-ts alignment.
Called by Laravel LocalStableTsDriver via subprocess.

Usage:
    python run_alignment.py <audio_path> <text_file_path> [--language yi] [--model ...]

Output: JSON to stdout with word-level timestamps
"""
import argparse, json, sys
from align import align_transcript_to_audio
from utils import get_breakable_align_model

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument('audio_path')
    parser.add_argument('text_path')
    parser.add_argument('--language', default='yi')
    parser.add_argument('--model', default='ivrit-ai/yi-whisper-large-v3-turbo-ct2')
    parser.add_argument('--device', default='auto')
    parser.add_argument('--compute-type', default='int8')
    args = parser.parse_args()

    with open(args.text_path, 'r', encoding='utf-8') as f:
        text = f.read().strip()

    model = get_breakable_align_model(args.model, args.device, args.compute_type)
    result = align_transcript_to_audio(
        audio_file=args.audio_path, transcript=text,
        model=model, language=args.language
    )

    # Convert WhisperResult to JSON
    words = []
    for seg in result.segments:
        if hasattr(seg, 'words') and seg.words:
            for w in seg.words:
                words.append({
                    'word': w.word.strip(),
                    'start': round(w.start, 3),
                    'end': round(w.end, 3),
                    'confidence': round(w.probability, 4) if hasattr(w, 'probability') else None,
                })

    print(json.dumps({'success': True, 'words': words, 'word_count': len(words)}))

if __name__ == '__main__':
    main()

  branch `feature/stable-ts-alignment`): Adds a new alignment driver using the
  ivrit-ai confusion-aware algorithm with the yi-whisper Yiddish model. Supports
  RunPod serverless GPU for production and local Python scripts for development.

  - CSV export of manual mappings
