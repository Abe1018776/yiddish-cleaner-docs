The Short Version

Three tools are being built to help preserve Yiddish audio recordings of the Lubavitcher Rebbe's teachings. Yiddish Cleaner (the biggest project, 100+ changes) is a web app that tests how well computers can transcribe Yiddish speech — it cleans up text, compares computer transcriptions against human ones, and scores accuracy. Audio-Text Mapping (brand new, built in one day) is a matching tool that pairs 4,669 audio recordings with their 1,065 written transcripts using smart suggestions. Audio Alignment (major new feature) takes an audio file and its transcript, then figures out exactly which word is spoken at which second — like karaoke subtitles for Yiddish recordings.

Yiddish Cleaner (KolYid)
A web application for testing and scoring how well different AI speech-to-text models handle Yiddish. Think of it as a "report card" system for Yiddish transcription software.
Laravel + PHP Vue.js 100+ commits 19 Pull Requests 16 Branches Jan 19 – Feb 26

What does this app do?

In everyday terms, here's what this tool lets people do:

🎤 Import Audio Recordings

Upload Yiddish audio files one at a time or in bulk from a Google spreadsheet. The app keeps track of every recording and its status.

📝 Manage "Ground Truth" Transcriptions

Attach human-verified written transcripts to audio files. These serve as the "correct answer" to compare against computer-generated transcriptions.

🧹 Clean Up Text

Automatically remove junk from transcripts (titles, editor notes, timestamps, brackets) using AI or rule-based cleaning. Comes with preset profiles for different eras of text formatting.

📊 Benchmark ASR Models

Run audio through different speech-to-text engines (Whisper, YiddishLabs, etc.) and compare their output against the correct transcript. Shows error rates and accuracy scores.

🔍 Word-by-Word Review

Review every single word the computer transcribed — see its confidence score, mark errors as "critical," delete bad words, or add missing ones.

🏆 Gold Standard Benchmarks

Flag the best, most carefully verified recordings as "gold standard" and show a special public leaderboard comparing how different AI models perform on them.

Two Systems in One Repo

This repository actually contains two related but distinct tools:

📄 Part 1: Transcript Cleaner (Original)

The older part of the project. A web app that takes raw Yiddish documents and strips out everything that isn't actual spoken words — titles, editor notes, timestamps, bracketed annotations, Hebrew citations. Supports Google Drive, Google Sheets batch processing, and AI-powered cleaning.

🎵 Part 2: Audio-Text Alignment (New)

The big new addition. Takes an audio file and its transcript, then produces word-level timestamps. Uses a special "confusion-aware" algorithm that detects when it's getting confused and intelligently skips ahead rather than producing garbage data. Runs on rented GPU servers.

How the Alignment Works (Simplified)

Step 1: You give it an audio file and a written transcript.
Step 2: The AI listens to the audio in chunks, trying to match what it hears to the written words.
Step 3: It gives each word a confidence score. When confidence drops (the "confusion zone"), it backs up and retries.
Step 4: If retrying doesn't help, it skips the confusing section and picks up where it can align again.
Step 5: The final result is every word tagged with its exact start and end time in the audio.
